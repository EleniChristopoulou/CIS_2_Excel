# CIS 2 Excel
This script extracts controls from a PDF file and exports them to Excel with formatting.

## Requirements

- Python 3.10+
- pip

## Installation

Clone the repository and install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Now simple run the script in directory it is located.

```bash
python noway.py
```

Then select you pdf and give it <sup>*TIME* :))</sup> to do its thnig.
